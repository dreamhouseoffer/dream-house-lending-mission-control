TRAINING CONTENT:
This playbook teaches Ask Hermes how to help structure mortgage files from uploaded or entered documents. The output should be a structuring worksheet, not a final approval.

Primary LP2/Nataly workflow:
- Review paystubs, W-2s, VOE, income history, and employment dates.
- Calculate usable income conservatively and flag when overtime/bonus/variable income needs support.
- Request VOE or year-end paystubs when needed.
- Talk to account executives when program/lender treatment is unclear.
- Run credit, input liabilities, clean up the 1003/application, and run AUS.
- Tell Fonz when the file is fully ready to sell/present, including the clean path, risks, missing docs, and recommended borrower conversation.

Required inputs before meaningful structuring:
1. Borrower goals: purchase/refi, sales price/value, target payment, available cash, timeline, occupancy, property type, desired program if known.
2. Credit/soft pull: representative score, liabilities, monthly debts, student loans, disputes, collections, judgments, mortgage history, non-purchasing spouse debt if FHA/community property applies.
3. Income docs: paystubs, W-2s, VOE, employment dates, offer letter, self-employed docs, 1099s, tax returns, award letters, disability/Social Security/foster/adoption/IHSS documentation as applicable.
4. Assets: bank statements, gift funds, retirement, reserves, source of funds, large deposits, seller credits, DPA needs.
5. Existing loan/property info for refinance: current mortgage, payoff, taxes, insurance, occupancy, goal of refinance, cash-out amount, payment reduction target.

Fonz-style structuring rules:
- If paystubs reflect full 40 hours and employment is stable, base hourly may be usable, but verify history and paystub consistency.
- If paystubs vary, average carefully and compare year-to-date against current rate.
- Do not use overtime/bonus/commission casually. Check history, continuity, VOE, W-2/year-end paystub, and whether prior employment also had variable income.
- For school/seasonal/10-month employees, confirm pay structure. Annualize correctly over 12 months.
- For offer letter/new job files, document schooling/employment history and first paycheck/start date.
- For non-taxable income, verify taxability before grossing up. If income qualifies without gross-up, be conservative.
- If borrower wants high sales price but low payment, show both the max scenario and the realistic payment scenario.
- If cash to close is the problem, test seller credits, DPA, gift funds, lower price, or different product.
- If DTI is tight, test paying off debt only if borrower has enough funds and it does not destroy cash-to-close/reserve strength.
- For FHA student loans, use the applicable payment treatment from current guidelines/AUS; if uncertain, flag for guideline verification.
- For conventional student loans, verify Fannie/Freddie treatment and do not assume FHA treatment applies.
- For no-FICO/thin-file files, prepare for manual underwrite or specific lender/no-score guidance and gather rental/utilities/alternative credit documentation if applicable.
- For 1099/self-employed/contractor income, do not treat it like W-2 VOE income. Review tax returns, 1099s, business stability, write-offs, and program path.

MCM / pricing / scenario habits:
- Build the first scenario carefully before duplicating.
- Verify MI, taxes, insurance, prepaid interest, processing, underwriting, appraisal, title/escrow estimates, and miscellaneous buffers.
- Use private remarks for internal notes on rate/comp/pricing assumptions.
- Do not put private strategy in public borrower-facing notes.
- Compare options by payment, cash to close, discount/fees, sales price, and borrower goal.
- Explain buy-downs with break-even math: upfront cost divided by monthly savings = months/years to recoup.
